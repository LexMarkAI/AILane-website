Fundraising — the Code behind every appeal

Why this matters to the Foundation

fundraising pays for the programmes, and the Code of Fundraising Practice (2025) sets the standard the public expects from every appeal, event and collection.

The duties
  • a written risk assessment for each fundraising event, kept as a record;
  • the licences and permissions the event needs (collections, alcohol, entertainment);
  • adequate insurance;
  • honest, pressure-free asks;
  • complaint handling with records.
